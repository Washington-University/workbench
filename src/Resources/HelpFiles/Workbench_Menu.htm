wb_view Menu

The wb_view Menu is exclusive to the Mac version and contains:

·         About wb_view… (Workbench version, compiling, and operating system information)

·         Preferences  

·         Mac-specific options which are available in all applications (Services, Hide Workbench, Hide Others, Show All)

·         Quit Workbench (to close the program)

In the PC and Linux versions, About wb_view…, Preferences, and Exit (to close the program) options appear in the File Menu.
