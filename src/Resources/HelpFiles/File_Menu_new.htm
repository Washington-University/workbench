File Menu

 

The File Menu appears slightly differently in the Mac version and PC/Linux versions due to Mac-exclusive Workbench Menu (contains About Workbench…, Preferences, and Quit Workbench options). On PC/Linux the File Menu contains:

 

·         About Workbench… to view WB version, compiling, and operating system information.

·         Preferences to set user specific Workbench configurations.

·         New Window to open a new Workbench Window (containing Menus, Tabs,  Toolbar and Toolbox)

·         New Tab to open a new viewing Tab within the active Workbench Window

·         Open File… to open any file in a Workbench readable format

·         Open Location… to open Workbench-readable files that are located remotely via a URL that the user sets (custom) or standard files through connection to ConnectomeDB (with login)

 

·         Open Recent Spec File… to open a previously opened specification file from a dropdown list

·         Save/Manage Files to save and/or manage options for all loaded files within one dialog.

o    Click the Save checkboxes to select files to be saved

o    A red YES in the Modified column warns you that a file has been modified and may need to be saved

o    The checkboxes in the In Spec column may be deselected/selected to modify the contents of a spec file. Remember to click the Save checkbox next to the Spec file to save these changes.

o    Files with a  button in the Read column are currently loaded into memory. Click the button to reload the file.

o    Files with a  button in the Read column are NOT currently loaded into memory. Click to load the file.

o    Clicking the button in the Remove column removes the file from memory.

o    Clicking the  button in the More column shows options to view/edit metadata and change the name of the file.

o    All changes made in the Save/Manage Files dialog must be confirmed by clicking the Save Checked Files button at the bottom right.

 

 

 

·         Close Spec File to close the spec file and all data that is currently loaded, without closing Workbench

·         Animation Control… to set controls for recording and playing movies in Workbench

o    Image Source sets the Workbench Window you are making the movie of.

o    Recording Control

·         Capture Image…

·         Close Tab to close the currently active Tab in the active Workbench Window

·         Close Window to close the currently active Workbench Window

·         Exit to close the program.
